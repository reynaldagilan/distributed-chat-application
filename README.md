# Distributed Chat Application

## Overview
A modern, asynchronous distributed chat application built with HTML5, CSS3, and JavaScript. This project demonstrates real-time messaging capabilities with a focus on responsive design and user experience.

## Features
- Real-time Messaging: Asynchronous communication using modern web APIs
- User Management: Create and manage user profiles
- Chat Rooms: Join or create multiple chat rooms
- Message History: Persistent storage of chat messages
- Responsive Design: Works seamlessly on desktop and mobile devices
- Notification System: Real-time notifications for new messages

## Project Structure
- index.html: Main HTML file
- styles.css: Styling and layout
- main.js: Main application logic
- chat.js: Chat functionality
- api.js: API communication module
- utils.js: Utility functions

## Getting Started
1. Clone the repository
2. Open index.html in your web browser
3. Create a username and start chatting

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Fetch API / WebSocket
